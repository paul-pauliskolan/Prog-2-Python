Kap 4.3 – Egna moduler och import

Organisera kod i moduler och paket.

Mål med lektionen

Efter denna lektion ska du kunna:

  • Skriva egna moduler och importera funktioner och klasser.
  • Förstå paketstruktur och `__init__.py`.
  • Använda relativa och absoluta imports samt `if __name__ == '__main__'`.

Skapa en modul

En modul är en .py-fil. Exempel: skapa `math_utils.py` med hjälpfunktioner.

# math_utils.py
def kvadrat(x):
    return x * x

def kub(x):
    return x * x * x

Importera från en modul

# i en annan fil
from math_utils import kvadrat
print(kvadrat(4))  # 16

import math_utils as mu
print(mu.kub(3))

Paket och __init__.py

En mapp med `__init__.py` är ett paket. Du kan gruppera moduler i paket och importera med punktnotation.

Relativa imports

# i paketets modul
from .helpers import parse_data  # relativ import från samma paket
from ..utils import logging_helper  # upp en nivå

Huvudvakt

Använd `if __name__ == '__main__'` för att köra testkod när modulen körs direkt.

def main():
    print("Testar modulen")

if __name__ == '__main__':
    main()

Tips för struktur

  • Dela upp funktionalitet i tydliga moduler.
  • Använd korta, återanvändbara funktioner.
  • Håll paketgränser rena och undvik cykliska imports.
