Kap 1.4 – Villkor och logik i funktioner

Använd if/elif/else för att fatta beslut i funktioner.

Så här lär du dig bäst

Gå igenom exemplen, skriv om dem och testa med olika indata. Fundera på ordningen i dina villkor och när en funktion ska returnera ett värde istället för att skriva ut något direkt.

Exempel – Kontrollera myndighet

def is_adult(age):
    if age >= 18:
        return True
    else:
        return False

user_age = int(input("Enter your age: "))
if is_adult(user_age):
    print("You are an adult.")
else:
    print("You are not an adult.")

def grade_feedback(grade):

        return "Excellent"
    elif grade >= 75:
        return "Good"
    elif grade >= 60:

Logiska uttryck

Ibland räcker det inte att kontrollera ett enda villkor. Ett program kan behöva kontrollera flera saker samtidigt. Då använder man logiska operatorer.

De tre vanligaste logiska operatorerna i Python är and, or och not.

and – båda villkoren måste vara sanna

and betyder att båda delarna måste vara True för att hela villkoret ska vara sant.

age = int(input("Ålder: "))
has_ticket = input("Har biljett? (ja/nej): ")

if age >= 18 and has_ticket == "ja":
    print("Du får gå in.")
else:
    print("Du får inte gå in.")

Programmet skriver bara Du får gå in. om personen både är minst 18 år och har biljett.

or – minst ett villkor måste vara sant

or betyder att det räcker att ett av villkoren är True.

day = input("Vilken dag är det? ")

if day == "lördag" or day == "söndag":
    print("Det är helg.")
else:
    print("Det är vardag.")

Här blir villkoret sant om dagen är lördag eller söndag.

not – vänder på ett villkor

not betyder inte. Det vänder ett villkor från True till False, eller från False till True.

logged_in = False

if not logged_in:
    print("Du måste logga in.")

Eftersom logged_in är False blir not logged_in sant.

Exempel kopplat till kap 2.7

I bibliotekskatalogen används or för att söka i antingen titel eller författare:

if query in book["title"].lower() or query in book["author"].lower():
    print("Boken matchar sökningen.")

Det betyder: om sökordet finns i titeln eller om sökordet finns i författaren, då räknas boken som en träff.

Du behöver inte förstå hela bibliotekskoden just nu. Poängen är att känna igen att logiska operatorer som or kommer tillbaka senare i kursen.

def check_entry(age, has_ticket, has_parent, is_blocked):
    # Skriv villkoren här.
    # Funktionen ska returnera rätt meddelande.
    pass

def main():
    age = int(input("Ålder: "))
    has_ticket = input("Har biljett? (ja/nej): ") == "ja"
    has_parent = input("Har vårdnadshavare med? (ja/nej): ") == "ja"
    is_blocked = input("Är personen avstängd? (ja/nej): ") == "ja"

    message = check_entry(age, has_ticket, has_parent, is_blocked)

main()

Programmet ska hantera

  • Ogiltig ålder: 0 eller negativa tal.
  • Avstängd person.
  • Person utan biljett.
  • Person som är minst 18 år och har biljett.
  • Person som är under 18 år men har biljett och vårdnadshavare med.
  • Person som är under 18 år, har biljett men saknar vårdnadshavare.
