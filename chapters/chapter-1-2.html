Programmering 2 i Python

Kap 1.2: Kom igång med Python

Mål med lektionen

  • Ha installerat Python och kunnat starta IDLE.
  • Veta hur du skriver, sparar och kör Python-program lokalt.
  • Ha skrivit och testat ett första program.
  • Veta varför du ska undvika att programmera i online-miljöer.

Så här lär du dig bäst

Följ installationsinstruktionerna noggrant. Använd alltid en lokal miljö – det vill säga ett program installerat på din dator. Undvik onlineverktyg (t.ex. repl.it eller liknande) eftersom de är begränsade och inte fungerar för hela kursens innehåll. Programmering på riktigt görs lokalt, precis som i arbetslivet.

För instruktioner om hur du installerar Python på macOS, se denna installationsguide för Mac. Följ också kapitel 1.7–1.10 i kursen Programmering 1 innan du fortsätter med övningarna här.

Du kan också repetera innehållet i Programmering 1 - Python om du vill fräscha upp grunderna.

Rekommendation: Vi rekommenderar att du använder VS Code som kodeditor, men du får använda IDLE också.

Skriv din första kod:

print("Hello, world!")

Spara filen som hello.py, tryck på F5 eller välj “Run” > “Run Module”.

Undvik detta

  • Använd inte online-lösningar som repl.it eller liknande.
  • Dessa fungerar inte med filhantering, UI-program, eller när du ska använda moduler som kräver lokalt stöd.

Reflektion

Du har nu installerat Python, startat en programmeringsmiljö och kört ett första program. Det kan verka enkelt, men det är ett viktigt steg: du har börjat ta kontroll över datorn som verktyg, inte bara som användare utan som utvecklare.
