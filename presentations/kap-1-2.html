Kap 1.2

Kom igång med Python

Vi repeterar hur du installerar, startar och kör Python lokalt.

Mål

Efter lektionen ska du kunna...

  • kontrollera att Python 3 finns installerat
  • starta IDLE och testa en kort instruktion
  • skriva och spara en Python-fil
  • öppna mappen i VS Code och köra programmet där

Installation

Installera Python 3

Gå till python.org.

Välj Downloads och ladda ner Python 3.

Öppna installationsfilen och följ stegen.

Python uppdateras ofta, så versionsnumret kan vara nyare än i exemplet.

Hitta Python

På en Mac

Öppna Finder.

Gå till Program eller Applications.

Leta efter en mapp som heter Python 3.x.

Du kan också använda Spotlight och söka efter Python eller IDLE.

Kontroll

Är Python installerat?

python3 --version

Visar vilken Python-version som datorn använder.

Python 3.13.7

Om du ser en Python 3-version är du på rätt väg.

IDLE

Starta Python enkelt

IDLE följer med när du installerar Python från python.org.

Starta IDLE med Spotlight: Cmd + mellanslag, skriv IDLE.

När IDLE öppnas ser du Python Shell.

Shell fungerar som en direkt testyta för Python.

Python Shell

Testa en instruktion direkt

print("Hej")

Du skriver en instruktion.

Hej

Python kör instruktionen och visar resultatet direkt.

Python-fil

Skriv ett program i en fil

print("Hej världen")

print() skriver text till skärmen eller terminalen.

"Hej världen"

Text inom citattecken är en sträng.

Spara

Python-filer slutar på .py

hej.py

Filändelsen .py visar att filen innehåller Python-kod.

Spara filen i din kursmapp.

Skapa gärna en mapp för kapitlet.

Använd tydliga filnamn, till exempel kap1-2-hej.py.

Kör i IDLE

Kör programmet från filen

  • Välj File och New File.
  • Skriv programmet i det nya fönstret.
  • Spara filen med .py.
  • Kör med F5 eller Run → Run Module.

VS Code

Installera och öppna kursmappen

Installera Visual Studio Code från code.visualstudio.com.

Starta VS Code.

Dra din kursmapp från Finder in i VS Code.

Då ser du dina Python-filer i sidomenyn.

VS Code

Installera Python-tillägget

Öppna Extensions i vänstermenyn.

Sök efter Python.

Installera Python-tillägget från Microsoft.

Tillägget hjälper VS Code att förstå Python-kod.

Interpreter

Välj rätt Python-version

Tryck Cmd + Shift + P.

Skriv Python: Select Interpreter.

Välj en version som börjar med Python 3.

Om fel version är vald kan programmet köras på fel sätt.

Kör i VS Code

Kör programmet

Öppna din .py-fil.

Klicka på Run-knappen uppe till höger.

Resultatet visas i terminalen längst ner.

python3 kap1-2-hej.py

I terminalen kan du även köra filen med python3.

Sammanfattning

Kom ihåg

  • Använd Python 3.
  • IDLE är bra för enkla första tester.
  • VS Code används när du arbetar med kursfiler.
  • Python-filer slutar på .py.
  • Spara allt i en tydlig kursmapp.
