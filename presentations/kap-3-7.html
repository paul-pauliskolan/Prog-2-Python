Objektorienterat projekt

Nu sätter vi ihop flera klasser till ett mindre projekt.

Mål

Efter lektionen ska du kunna...

  • planera ett objektorienterat program
  • använda flera samverkande klasser
  • rita ett klassdiagram
  • strukturera kod i flera delar
  • testa klasser innan helheten byggs ihop

Projekt

Vad ska byggas?

Ett mindre Pythonprojekt.

Minst tre klasser som samarbetar.

Tydlig struktur, UML-diagram och fungerande kod.

Projektidéer

Välj ett område

  1. Bibliotekssystem: böcker, låntagare och bibliotek.
  2. RPG-strid: spelare, fiender och föremål.
  3. Receptplanerare: recept, ingredienser och inköpslistor.

Arbetsgång

Bygg i fem steg

  1. Designa klassdiagrammet.
  2. Skapa grundstrukturen.
  3. Lägg till logik och relationer.
  4. Testa objekt och metodanrop.
  5. Dokumentera hur programmet fungerar.

Planering

Frågor innan kod

  • Vilka objekt finns i problemet?
  • Vilka attribut behöver varje objekt?
  • Vilka metoder ska varje klass ha?
  • Vilka klasser känner till varandra?
  • Finns det arv, eller bara association?

Exempel

Bibliotekssystem

En bok kan lånas och lämnas tillbaka.

En medlem kan låna böcker.

Biblioteket håller reda på böcker och medlemmar.

Relationer

Vad visar diagrammet?

  • Library innehåller böcker.
  • Library registrerar medlemmar.
  • Member kan låna böcker.
  • Relationerna är associationer, inte arv.

Book

Ansvar

  • Spara titel och författare.
  • Hålla reda på om boken är tillgänglig.
  • Lånas ut med borrow().
  • Lämnas tillbaka med return_book().

Book

Kod

class Book:
    def __init__(self, title, author):
        self._title = title
        self._author = author
        self._available = True

    def borrow(self):
        if self._available:
            self._available = False
            return True
        return False

    def return_book(self):
        self._available = True

Member

Ansvar

  • Spara medlemmens namn.
  • Spara lånade böcker i en lista.
  • Låna bok genom att anropa book.borrow().
  • Lämna tillbaka bok genom att anropa book.return_book().

Member

Kod

class Member:
    def __init__(self, name):
        self._name = name
        self._borrowed_books = []

    def borrow_book(self, book):
        if book.borrow():
            self._borrowed_books.append(book)
            print(f"{self._name} lånade {book.get_title()}.")
        else:
            print(f"{book.get_title()} är redan utlånad.")

Library

Ansvar

  • Spara alla böcker.
  • Spara alla medlemmar.
  • Lägga till böcker och medlemmar.
  • Hitta bok eller medlem med sökning.

Library

Kod

class Library:
    def __init__(self):
        self._books = []
        self._members = []

    def add_book(self, title, author):
        book = Book(title, author)
        self._books.append(book)

    def register_member(self, name):
        member = Member(name)
        self._members.append(member)

main.py

Sätt ihop projektet

lib = Library()

lib.add_book("1984", "George Orwell")
lib.register_member("Oskar")

book1 = lib.find_book("1984")
member = lib.find_member("Oskar")

if book1 and member:
    member.borrow_book(book1)
