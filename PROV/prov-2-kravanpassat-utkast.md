# Prov 2 - Kravanpassat utkast

50 flervalsfrågor. Varje fråga har fyra alternativ och exakt ett rätt svar.

Fråga 41-50 är svåra kodfrågor.

### 41. Vad skrivs ut?
```python

summa = 0

    if t % 2 == 0:
        summa += t

print(summa)
```

B. `10`  
C. `13`  
D. `[4, 6, 2]`
