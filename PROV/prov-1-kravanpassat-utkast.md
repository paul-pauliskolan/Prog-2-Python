# Prov 1 - Kravanpassat utkast

50 flervalsfrågor. Varje fråga har fyra alternativ och exakt ett rätt svar.

Fråga 1-20 fokuserar främst på grundkunskaper och enkla problem.  
Fråga 21-40 fokuserar mer på problemlösning, struktur, användare, tillgänglighet och val av lösning.  
Fråga 41-50 är svåra kodfrågor.

## Svåra kodfrågor 41-50

### 41. Vad skrivs ut?
```python
poang = [3, 7, 2]
högst = poang[0]

for p in poang:
    if p > högst:
        högst = p

print(högst)
```
A. `7`
B. `2`
C. `3`
D. `[3, 7, 2]`

**Rätt svar: A**

### 42. Vad skrivs ut?
```python
def kontrollera(alder):
    if alder >= 18:
        return "myndig"
    return "inte myndig"

print(kontrollera(17))
```
A. `myndig`
B. `inte myndig`
C. `17`
D. `None`

**Rätt svar: B**

### 43. Vad skrivs ut?
```python
namn = ["Ali", "Bo", "Cia"]

for n in namn:

        resultat.append(n)

print(resultat)
```

B. `["Ali", "Bo", "Cia"]`
C. `["Ali", "Cia"]`
D. `[]`

**Rätt svar: C**

### 45. Vad skrivs ut?
```python
class Bok:
    def __init__(self, titel, sidor):
        self.titel = titel
        self.sidor = sidor

    def kort_info(self):
        return self.titel + " har " + str(self.sidor) + " sidor"

bok = Bok("Python", 120)
print(bok.kort_info())
```
A. `Python har 120 sidor`
B. `Bok har 120 sidor`
C. `titel har sidor`
D. `None`

**Rätt svar: A**
